安装依赖环境

Hexo 是基于 Node.js 的静态博客框架，虽然个人不太了解 nodejs，不过相关步骤还是比较简单的，照着网上的攻略走就行。在搭建之前，需要确保本地已经安装了 Node.js 和 npm（Node.js 的包管理工具）。以下是具体的安装步骤：

安装 Node.js 和 npm
